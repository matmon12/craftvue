    ## Подключение

    По умолчанию компоненты импортируются без стилизации. Стили подключаются отдельно в `main.ts(js)`.

    <CustomSource
      code={configCodeTexts.styles}
      actions={['copy']}
      parser="babel"
      language="typescript"
    />

    &nbsp;

    ## Типографика

    Изначально для всех компонентов задан следующий набор шрифтов:

    <CustomSource
      code={configCodeTexts.themingFont}
      actions={['copy']}
      parser="html"
      language="html"
    />

    Если необходимо использовать в проекте стандартный шрифт `Montserrat`, его нужно подключить в файле со стилями:

    <CustomSource
      code={configCodeTexts.themingFontSetup}
      actions={['copy']}
      parser="html"
      language="html"
    />

    Перед подключением шрифт необходимо скачать и поместить в соответствующую директорию. Рекомендуется использовать формат `woff2`.

    <Typeset
      fontSizes={[
        typography.size.xxs,
        typography.size.xs,
        typography.size.s,
        typography.size.m,
        typography.size.l,
        typography.size.xl,
        typography.size.xxl,
      ]}
      fontWeight={typography.weight.regular}
      sampleText='Использование шрифта Montserrat'
      fontFamily={typography.type.primary}
    />

    ## Цвета

    ### Основные

    Все цвета в библиотеке заданы в css переменных. Базовые цвета библиотеки определяют переменные ниже. Основной цвет записан в переменной `--prime-color`.

    <CustomSource
      code={configCodeTexts.themingColorBasic}
      actions={['copy']}
      parser="html"
      language="tsx"
    />

    <Story of={PaletteStories.Basic} className="sb-docs-palette" />

    ### Компоненты

    Для каждого компонента задан отдельный набор переменных, определяющих его палитру. Ниже приведен набор переменных для компонента `CBadge`.

    <CustomSource
      code={configCodeTexts.themingColorComponent}
      actions={['copy']}
      parser="html"
      language="tsx"
    />

    ### Палитра

    Основные переменные и переменные комопнентов формируются на основе цветов из базовой палитры.

    <CustomSource
      code={configCodeTexts.themingColorPalette}
      actions={['copy']}
      parser="html"
      language="html"
    />

    <Story of={PaletteStories.Palette} />

    &nbsp;

    ## CSS-переменные

    Для задания некоторых стилей используются CSS-переменные. Кроме цветов переменные используются также и для других свойств: `border-radius` и `z-index`.

    <CustomSource
      code={configCodeTexts.themingCssProperties}
      actions={['copy']}
      parser="html"
      language="html"
    />

    Полный набор переменных хранится в файле `_vars.scss`. При подключении стилей все переменные проекта становятся также доступными в проекте.
