    ## Скачивание

    Для загрузки библиотеки иконок в проект, используете следующую команду.

    <CustomSource
      code={configCodeTexts.iconsDownload}
      actions={['copy']}
      parser='bash'
      language="apache"
    />

    &nbsp;

    ## Импорт

    Для использования необходимо имортировать необходимые иконки из библиотеки. Чтобы не подключать всю библиотеку, можно использовать импорт иконок по отдельности.

    <CustomSource
      code={configCodeTexts.iconsImport}
      actions={['copy']}
    />

    &nbsp;

    ## Компонент [Icon](?path=/docs/components-icon--docs)

    Для более удобного и гибкого использования библиотека интегрирована в компонент [Icon](?path=/docs/components-icon--docs) из CraftVue. В атрибут `name` необходимо передать название нужной иконки, напимер `home`. Для задания нужного цвета и размера комопнент предоставляет соответствующие атрибуты.

    <Canvas of={IconStories.Color} sourceState="none" className="sb-canvas" />

    <CustomSource type="icon" name="color" />

    &nbsp;
